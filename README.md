# 📌 Project Overview

The IPL auction is one of the most critical decision-making events for every franchise. Selecting the right players requires balancing current squad composition, player performance, overseas player limits, team requirements, and budget constraints.

This project simulates a **Mumbai Indians Auction War Room**, where historical IPL and domestic cricket data are transformed into an interactive Business Intelligence dashboard that supports strategic auction planning.

The dashboard enables coaches, analysts, and management to identify squad gaps, evaluate player performance, monitor team balance, and shortlist auction targets using data-driven insights instead of intuition.

---

# 🎯 Business Problem

Franchises often evaluate hundreds of players before an IPL auction. Without a centralized analytics platform, comparing players across multiple seasons and identifying team weaknesses becomes difficult.

This project addresses that challenge by providing:

- Squad composition analysis
- Team balance visualization
- Batting performance analytics
- Bowling performance analytics
- Domestic talent scouting (SMAT)
- Historical team performance trends
- Interactive filtering for strategic decision-making

---

# 🎯 Objectives

- Build an executive dashboard for franchise management.
- Analyze Mumbai Indians' current squad composition.
- Evaluate batting and bowling performance using historical IPL data.
- Identify potential domestic talent using Syed Mushtaq Ali Trophy (SMAT) statistics.
- Support data-driven auction planning.

# 📂 Dataset

The project combines multiple datasets including:

- Historical IPL batting statistics
- Historical IPL bowling statistics
- Mumbai Indians squad information
- Match results
- SMAT batting statistics
- SMAT ball-by-ball data

The raw datasets were cleaned, transformed, and merged using Python before visualization in Power BI.
